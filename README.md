<h1>iChat – Simple Web Chat App 💬</h1>
<h3>📌 Short Description</h3>

iChat is a simple WhatsApp-style web chat interface built using HTML, CSS, and JavaScript. It simulates a messaging environment where users can send messages and hear notification sounds, providing a basic real-time chat experience in the browser.<br><hr>

<h3>📖 Project Description</h3>

iChat is a lightweight web chat application designed to demonstrate the core concepts of front-end web development. The project focuses on building a clean messaging interface similar to popular chat applications.<br>

This project includes a chat UI, message sending functionality, notification ringtone, and responsive layout. It is built using only vanilla HTML, CSS, and JavaScript, making it ideal for beginners who want to understand how chat interfaces work without using complex frameworks.<br>

The application also includes custom assets such as a PNG logo and a notification ringtone, making the interface feel more interactive and realistic.<br><hr>

<h3>🚀 Features</h3>
<li>💬 Send and display chat messages</li>
<li>🔔 Message notification ringtone</li>
<li>🖼 Custom logo integration</li>
<li>📱 Responsive chat interface</li>
<li>🎨 Clean and simple UI design</li>
<li>⚡ Built with pure HTML, CSS, and JavaScript</li>
<hr>
<h3>🛠️ Technologies Used</h3>
<li>HTML5 – Structure of the application</li>
<li>CSS3 – Styling and layout design</li>
<li>JavaScript (Vanilla JS) – Chat functionality and interactions</li>
