<h1>iChat - Real-time Chat Application</h1>
iChat aik dynamic real-time messaging platform hai jo fast aur seamless communication provide karta hai. Is project ko Node.js aur Socket.io ki madad se banaya gaya hai taake users bina kisi delay ke aik dusray se connect ho sakain.

<h2>✨ Features</h2>
Real-time Messaging: Messages foran deliver hotay hain bina page refresh kiye.

Responsive UI: Mobile aur Desktop dono par behtareen chalti hai.

User Notifications: Jab koi naya user join karta hai ya message bhejta hai, to notifications show hoti hain.

<h2>🛠️ Tech Stack</h2>
Frontend: HTML5, CSS3, JavaScript (Vanilla).

Backend Support: Socket.io for real-time bi-directional communication.

<h2>🚀 How to Run</h2>
Is repository ko clone karain.

index.html file ko kisi bhi modern browser mein open karain.

Ensure karain ke aapka socket server backend par run ho raha ho.
